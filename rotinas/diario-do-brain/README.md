# Rotina diária — diário do brain

## Status
Não configurada por padrão.

## Objetivo
Fechar o dia com um resumo executivo do que realmente importou: decisões, aprendizados, riscos, sinais e próximo passo recomendado.

## Entradas recomendadas
- sessões recentes
- estado do git do repositório
- mudanças relevantes em projetos vivos
- sinais de operação que merecem registro durável

## Saída esperada
- atualização no diário
- mensagem curta no canal definido
- zero dump bruto de conversa

## Regras
- não duplicar backlog
- não inventar fatos
- se o dia não produziu contexto durável, dizer isso claramente
