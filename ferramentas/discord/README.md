# Discord

## Papel
Cockpit operacional com canais, threads e entrega de rotinas.

## Quando faz sentido
- operação com múltiplos agentes
- times que precisam de canal por frente
- rotinas com histórico e threads claras

## O que decidir
- canal home
- canais específicos por operação
- se cada agente terá canal/thread própria
- quem pode acessar o quê

## Riscos
- excesso de canais
- mistura de operação com conversa informal
- permissões mal definidas
