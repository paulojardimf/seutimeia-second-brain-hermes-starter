# Workers de código e leitura

## Papel
Workers externos ou agentes delegados para leitura pesada, varredura ampla, research ou execução paralela.

## Exemplos de uso
- leitura pesada de múltiplos arquivos
- revisão de mudanças grandes
- comparação de alternativas
- coding workers especializados

## Regra
O worker pode ler ou executar, mas a curadoria final continua no agente principal.
