# Google Workspace

## Papel
Gmail, Calendar, Drive, Docs e Sheets como camada de comunicação e documentos vivos.

## Casos comuns
- varredura de inbox operacional
- agenda do dia
- documentos e planilhas compartilhadas
- repositório de anexos e materiais de cliente

## Risco
Virar silo paralelo se não houver fonte canônica definida.
