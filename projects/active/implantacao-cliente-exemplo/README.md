# Implantação cliente — exemplo

## O que é
Exemplo de estrutura para clientes que receberão operação de agentes, segundo cérebro e rotinas guiadas.

## O que documentar
- contexto do cliente
- ferramentas reais disponíveis
- restrições de acesso
- formato de entrega
- canais de comunicação
- projetos e automações prioritárias

## Regra
Nunca colocar dado sensível do cliente sem governança explícita.
