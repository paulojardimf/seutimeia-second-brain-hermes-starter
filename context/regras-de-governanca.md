## Regras operacionais
1. README por pasta relevante.
2. Estrutura e contexto durável ficam aqui; backlog fica na ferramenta de execução.
3. Credenciais nunca entram no repositório.
4. Conteúdo sensível deve ser resumido ou referenciado, não exposto.
